Fly Through (3D Mobile Game)

I started making this game to challenge myself so that I can furher improve as a game developer. I was inspired by Geometry Dash and Subway Surfers and decided to make a 3D airplane game. The goal of this game is to dodge all of the obstacles and gets to the portal across the map.
Some of the challenging things I had to implement was:

  • Circular Doubly Linked List to select airplane skins
  • Singleton for Game Manager
  • Managing Levels

Fruits Bomber (2D Mobile Game)

In crafting my first published game, my primary goal was to transition from private game development to creating a fully polished product for public enjoyment. Target audience for this game was kids, ensuring both entertainment and simplicity. This endeavor marked a pivotal moment in my journey as a game developer.
Some of the things I learned:

  • Managing scores
  • Drag and Drop on Mobile
  • Implmentation of interstitial, banner, and rewarded advertisement
  • The process of releasing a game to Google Play
