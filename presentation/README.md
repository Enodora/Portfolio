# Nintendo Interview Presentation

A React-based slide presentation for game development portfolio showcase, styled with a clean Nintendo-inspired white and red theme.

## Features

- 7 slides covering portfolio projects
- Smooth slide transitions with Framer Motion
- Keyboard navigation (Arrow keys, Spacebar)
- Progress bar and slide counter
- Responsive design
- Japanese language content

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

The presentation is automatically deployed to GitHub Pages via GitHub Actions when changes are pushed to the `main` branch. It will be available at:

**atsuhiroshirakaki.com/presentation/**

## Slide Structure

1. **Title Slide** - Introduction with name and title
2. **Intro Slide** - Self-introduction and university background
3. **UBCRoid Slide** - Project showcase with gameplay image and video
4. **Fruits Bomber Slide** - Project showcase with gameplay image and video
5. **砕幻 Overview Slide** - Project overview with title logo
6. **砕幻 Systems Slide** - Key systems breakdown with demo video
7. **Closing Slide** - Contact information and social links

## Navigation

- **Arrow Right** or **Spacebar**: Next slide
- **Arrow Left**: Previous slide
- **Navigation buttons**: Click to navigate
